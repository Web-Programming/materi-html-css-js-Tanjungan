Belajar HTML

HTML adalah bahasa markup yang digunakan untuk membuat halaman web. HTML terdiri dari berbagai elemen yang digunakan untuk menyusun dan mengatur konten pada halaman web.

Elemen HTML

  • <h1> - <h6>: Digunakan untuk membuat judul dengan tingkat kepentingan yang berbeda.
  • <p>: Digunakan untuk membuat paragraf.
  • <a>: Digunakan untuk membuat tautan (link) ke halaman lain atau sumber daya lainnya.
  • <img>: Digunakan untuk menyisipkan gambar ke dalam halaman web.
  • <div>: Digunakan untuk mengelompokkan elemen-elemen lain
